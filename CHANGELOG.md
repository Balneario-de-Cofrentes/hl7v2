# Changelog

## v0.1.0 (unreleased)

- Initial release
- Raw and typed parsing modes
- Typed segments: MSH, EVN, PID, PV1, NK1, OBR, OBX, ORC, MSA, ERR, NTE, AL1, DG1
- Data types: ST, NM, DT, DTM, TS, ID, IS, SI, TX, FT, CX, XPN, XAD, XTN, CE, CWE, HD, PL, EI, MSG, PT, VID
- Message builder API
- ACK/NAK builder
- MLLP framing, listener (Ranch 2.x), and client
- TLS/mTLS support
- Telemetry events
- Validation engine (structure + field-level)
